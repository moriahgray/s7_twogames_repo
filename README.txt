# Two Games

## Overview

**Two Games** is a web-based application featuring two interactive games: _Coin Toss_ and _Guessing Game_. It's designed to provide users with simple yet engaging games, all within a user-friendly interface.

### Features

- **Coin Toss Game**: Flip a coin and keep track of wins and losses.
- **Guessing Game**: Guess a random number between 1 and 100 in 10 tries.

## Installation

Clone the repository to get started:

1. Clone the repository:
   git clone https://github.com/moriahgray/s7_twogames_repo.git

## How to Run

Running **Two Games** is straightforward:

1. Open your preferred web browser.
2. Use the browser to navigate to the folder where you cloned the repository.
3. Locate and open the `index.html` file in the browser.
